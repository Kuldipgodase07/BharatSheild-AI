# BharatShield AI Notebooks

This directory contains Jupyter notebooks for exploratory data analysis (EDA), 
model training, and evaluation for the BharatShield AI fraud detection platform.

## Structure

| Notebook | Description |
|---|---|
| `01_EDA.ipynb` | Exploratory Data Analysis on insurance claims dataset |
| `02_feature_engineering.ipynb` | Feature engineering and preprocessing pipeline |
| `03_model_training.ipynb` | Training the fraud detection ML model |
| `04_model_evaluation.ipynb` | Model evaluation, ROC curves, confusion matrix |
| `05_deepfake_detection.ipynb` | Document deepfake/forgery detection experiments |

## Setup

```bash
pip install jupyter pandas numpy scikit-learn matplotlib seaborn
jupyter notebook
```
